## CV/CA Prediction Model

The CV/CA prediction code in this project uses the multilingual Valence-Arousal prediction model from the following GitHub repository:

- Model repository: https://github.com/gmendes9/multilingual_va_prediction

This repository provides the implementation for *Quantifying Valence and Arousal in Text with Multilingual Pre-trained Transformers*, including model training and prediction-related code for estimating valence and arousal from text.
